# 📖 School Library <a name="about-project"></a>

**School Library** is a is a powerful Ruby-based application designed to efficiently manage the library's book inventory and track borrowing activities. With its intuitive interface, it allows librarians to effortlessly record and update the availability of books in real-time, ensuring accurate inventory management. Additionally, students can easily borrow books through the app, enabling seamless tracking of borrowing history and enhancing the overall library experience.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
  Open (https://github.com/microverseinc/curriculum-ruby/blob/main/simple-ruby/articles/ruby_installation_instructions.md) in your browser to install ruby on your prefered OS
```
### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:CaptainBawa/School-Library-App.git
```
### Install

Install this project with:

```sh
  cd my-project
  gem install
```
### Usage

To run the project, execute the following command:

```sh
  rib
```
### Run tests

To run tests, run the following command:

```sh
  rubocop
```
